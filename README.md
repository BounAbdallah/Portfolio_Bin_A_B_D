# Portfolio - Bouna Bin Abdallah Dramé

Bienvenue sur le code source de mon portfolio professionnel.
Je suis un développeur full-stack basé à Dakar, Sénégal, spécialisé dans :
- **Frontend** : React, Next.js, Tailwind CSS
- **Backend / ERP** : Laravel, Node.js, Python/Django
- **Mobile** : React Native

## Technologies du portfolio
Ce projet a été conçu avec :
- [Next.js](https://nextjs.org/) (App Router)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/) pour les animations
- [Lucide React](https://lucide.dev/) pour les icônes

## Pour lancer le projet
```bash
# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev
```

Visitez [http://localhost:3000](http://localhost:3000) pour voir le résultat.
